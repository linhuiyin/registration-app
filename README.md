# 📋 报名助手 - 免费报名小程序

一个完全免费、即开即用的扫码报名工具。生成二维码后用微信或其他扫码软件扫描即可报名。

## ✨ 功能特点

- 🆓 **完全免费** — 无需服务器、无需数据库，纯前端运行
- 📱 **微信扫码报名** — 生成二维码，微信一扫就能填表报名
- ⚡ **零部署门槛** — 单 HTML 文件，拖到任何静态托管即可使用
- 🔒 **数据本地存储** — 报名数据保存在浏览器 localStorage，无需后端
- 📊 **数据导出** — 一键导出报名数据为 CSV（Excel 可打开）
- 🎨 **美观易用** — 现代化 UI，手机端体验流畅
- 📝 **自定义字段** — 自由设置报名字段（文本、电话、邮箱、选择等）
- 👥 **名额限制** — 支持设置报名上限和截止日期

## 🚀 免费部署方式

### 方式一：GitHub Pages（推荐）

1. 创建 GitHub 仓库
2. 将 `index.html` 上传到仓库
3. 进入 Settings → Pages → Source 选择 main 分支 → Save
4. 等几分钟即可通过 `https://你的用户名.github.io/仓库名/` 访问

### 方式二：Vercel

1. 访问 [vercel.com](https://vercel.com) 注册（用 GitHub 账号）
2. 导入项目 → 选择文件夹 → Deploy
3. 自动获得 `https://xxx.vercel.app` 域名

### 方式三：Netlify

1. 访问 [netlify.com](https://netlify.com) 注册
2. 拖拽 `index.html` 文件到页面即可部署
3. 自动获得 `https://xxx.netlify.app` 域名

### 方式四：Cloudflare Pages

1. 访问 [pages.cloudflare.com](https://pages.cloudflare.com)
2. 连接 GitHub 仓库或直接上传文件
3. 自动获得 `https://xxx.pages.dev` 域名

## 📖 使用教程

### 1. 创建活动
点击「+ 新建」按钮，填写活动信息：
- 活动标题（必填）
- 活动描述
- 报名截止日期
- 报名人数上限
- 自定义报名字段（姓名、手机号、单位等）

### 2. 分享报名
创建活动后自动生成二维码：
- 📱 截图分享给参与者扫码
- 📋 复制链接发送到微信群
- 📤 支持系统分享

### 3. 参与者报名
参与者扫描二维码后：
1. 打开报名表单页面
2. 填写报名信息
3. 提交完成报名 ✅

### 4. 查看/导出数据
在活动详情页：
- 查看所有报名记录
- 搜索筛选报名信息
- 一键导出 CSV 文件

## ⚠️ 注意事项

- 数据存储在浏览器中，**清除浏览器数据会导致记录丢失**
- 建议定期导出 CSV 备份
- 不同设备/浏览器之间的数据不互通
- 如需多人协作管理，建议部署后共用同一设备查看

## 🛠 技术栈

- 纯 HTML/CSS/JavaScript，无框架依赖
- QRCode.js CDN 生成二维码
- localStorage 数据持久化
- PWA 友好（可添加到手机主屏幕）
